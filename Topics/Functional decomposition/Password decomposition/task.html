Password decomposition

You have the following code:

fun main() {
    var password = readln()
    if (password.length < 5) {
        println("Your password should be longer than five characters, please write a new password")
        val newPassword = readln()
    } else {
        println("Your password is correct, please enter your name")
        val name = readln()
        println("Congratulations! Password for $name is $password")
    }
}

Decompose the program to the functions passwordLessThan5() and passwordLongerThan5(password: String). In your new functions, replace readln() with a random password longer than five characters in newPassword and with John in name. Don't change println() in your functions!
